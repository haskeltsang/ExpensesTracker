# How to use

## Install docker and docker-compose
```
https://www.docker.com/
```
## Copy prod.env to .env and change your own variables
```
cp prod.env .env
```

## Run the following command
```
docker compose up -d
```

## Open your browser and go to the following link
```
http://localhost
```
